Welcome to the HSDAPI AirTable Portal

This portal shows you how to interact with AirTables that have adopted Open Referral's Human Services API Standard. AirTable is a spreadsheet-like tool that allows users to create, manage and share multi-table datasets and make them available via API. Open Referral is the leading open data standard for health, social and human services in the USA. Sahana Software Foundation created an AirTable Template for the Open Referral's data standard. API Evangelist created scripts that makes the API generated by the AirTable Open Referral Template compatible with the Open Refertal API Standard. We're all proud to present this portal to you.

Authentication

This site requires using an API key which is issued as part of the AirTable configuration. If you do not hav an API key you won't be able to pull any data from the API.

  • api_key - The API key.

It only takes one key for now, which is issued by AirTable--we can do more later, if more than one key is needed for a varietiy of applications.
